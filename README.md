# Sparta-Nginx-Reverse-Proxy

## The task

To use an Nginx reverse proxy to make the node port the root for our IP address

## To Use

Clone the repo down and run vagrant up. Then type

`vagrant ssh app`
`cd /vagrant/app`
`pm2 start app.js`

Fianlly, go to [this page](http://development.local) to see the app and [this page](http://development.local/posts) to see the database.
